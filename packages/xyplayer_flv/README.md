### 使用


启动： **yarn dev**

访问： [http://localhost:3000](http://localhost:3000)

编译： **yarn build**



---

### 说明

编译输出： CJS/LIFE/UMD/ESM

默认使用： UMD 。
-  可以根据情况修改**package.json** main设置默认使用文件。

---


### 开发配置

端口修改：

文件：**rollup.config.js**

- port: 3000
